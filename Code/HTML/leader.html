Microsoft
Microsoft has been well known for their involvement in all three types of cloud computing. Microsoft is also known for their involvement in artificial intelligence, and the software part of cloud computing called "the intelligent cloud." Microsoft has been focusing more on cloud computing with Azure and Office 365, and incorporating "the intelligent web" into their cloud computing products and services.
Amazon Web Services (AWS)
Amazon has been in this business since 2002, when it started its cloud based service, AWS. Since then, it has created the first IaaS platform and has gone even further with its shopping website. Amazon also has created multiple IaaS and SaaS services such as Elastic Compute Cloud (EC2), Elastic Beanstalk, Simple Storage Service (S3), and Elastic Block Store (EBS).
Salesforce
Salesforce and Microsoft lead a fast growing branch of cloud computing, SaaS. Salesforce is a Customer Relationship Management (CRM) Platform that specializes in cloud-based applications for sales, service, and marketing. They specialize in customer managment and satisfation. Salesforce needs no IT to run, and you can set up and go.
Google Cloud Platform
Google Cloud Platform is a service that uses the same infrastructure as regular G-Suite services like Youtube, Gmail, and Drive. This service uses AI, machine learning, data analysis, and more. It has lots of products like the Compute Engine (IaaS which provides Virtual machines), BigQuery (IaaS for large data analysis), and more. Google Cloud Platform offers more types of services than AWS and Azure.

Jobs and Careers

Software Engineer

Software engineers design and create the code required to run applications. They’re the ones who fix bugs or add new features. Software engineers are expected to be effective collaborators, be able to efficiently test software, and have decent analytic skills. Cloud computing is wholly dependent on the success of these engineers since they create the very platforms that cloud computing is built on.

Data Engineer

A data engineer is the gateway to business insight. Data engineers build the data infrastructure to answer business questions and improve marketing strategies. They make, maintain, and construct scalable data management systems. Data engineers also make high performance algorithms and integrate software into the data infrastructure.

System Administrator

The System Administrator maintains various IT systems. This may cover system provisioning, configuration, maintenance, and decommissioning. The System Administrator, “is the nuts and bolts” of the system, making sure the hardware and software is working and up to date. The System Administrator is responsible for maintaining web apps, emails, security, and more. They are the go-to when a client has an issue or something goes wrong.
