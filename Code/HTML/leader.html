Amazon Web Services (AWS)
Amazon is a major leader in cloud computing. It has been in this business since 2002, when it started its cloud based service, Amazon Web Services (AWS). Since then, it has created the first IaaS platform and has gone even further with its shopping website. Amazon also has created multiple IaaS and SaaS services such as Elastic Compute Cloud (EC2), Elastic Beanstalk, Simple Storage Service (S3), and Elastic Block Store (EBS). Plus, its services reach even to Artificial intelligence, which they call Amazon Alexa. Alexa searches through the cloud for information, and can do many functions because of this.

Google Cloud Platform
Google Cloud Platform is a cloud computing service that uses the same infrastructure regular G-Suite services, like Youtube, Gmail, and Drive. This service uses AI, machine learning, data analysis and more. Google Cloud Platform uses all basic branches of cloud computing and a little bit more. It has lots of products like the Compute Engine, that is a Iaas providing Virtual machines, BigQuery for large data analysis, and more. Google Cloud Platform is a very well rounded cloud computing service that could be compared to many other services like Microsoft Azure, and Amazon Web Services.
