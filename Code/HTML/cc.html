Definition

Cloud computing is the delivery of computing services wirelessly through the internet, or the cloud. This allows data to be transferred effortlessly from one source to another, or be safely secured. For example, when someone wants to get images from a computer to a phone, they put the pictures on the cloud and access them from wherever, whenever. Cloud computing makes it so that you don’t have to download additional software or hardware, so that all you have to do is rent it from your provider.

SaaS

Infrastructure as a Service is the basic physical and organizational structures and facilities needed for the operation of a society or enterprise. Infrastructure as a Service is when you rent an infrastructure like a server, storage, operating system, etc., from a wireless provider. Some IaaS examples are servers and storage, networking firewalls and security, a data center, and a physical plant or building.

Platform as a Service is a bit broader than Infrastructure as a Service, since Infrastructure as a Service is part of Platform as a Service. Platform as a Service is a Cloud computing service based off of a computer platform such as google docs. It is used with operating systems, development tools on a computer, database management, business analytics, and all of the uses listed under Infrastructure as a Service. Private PaaS servers Allow people to manage applications while still keeping it’s privacy. Some companies that use private PaaS servers are Google, Amazon, and Windows.

Software as a Service is like a bank. It protects the valuables and privacy of its customers, while providing reliable service. Software as a Service is a service where something is downloaded as a software onto a device. This counts as apps, launchers, and all of the uses listed in Platform as a Service and Infrastructure as a Service. Software as a Service is also a way of transmitting applications over the Cloud, Hence the word Software in its name. Because of this, Software as a Service Frees up time and hassle from managing Software and hardware. Some advantages of this service are its Painless and hassle free Upgrades, its low initial cost, it is integrated into the computer system seamlessly, and it is adopted by many.

Deployments

Hybrid Cloud
The hybrid cloud is a combination of the Private cloud and public cloud, and is made up of one or more public cloud providers. It allows data to move between private and public clouds, which enables more flexibility and deployment. The hybrid clouds can also manage security issues, and is very efficient.
Public Cloud
The Public cloud is a cloud service that is owned and operated by a third-party provider. In this type of cloud service, the provider has complete management over the service. Though the public, Public Cloud has a reasonable amount of security, it’s not the best option if security is a concern.
Private Cloud
The Private Cloud is a cloud service that is usually used by a single business or organization. It can be located in a physical form at the companies that provide it, and has greater control over the cloud infrastructure. The Private Cloud also has the highest levels of reliability and scalability, although its infrastructure is limited to the private network.

File storage

Cloud is used for File storage when someone wants to store files without taking up space on the computer. To do this, they would put the files on a private cloud, and store the pictures there.

Developing

Cloud computing is used for Developing when businesses need to secure a budget for the year. Using a cloud service with google, the business would be able to create the budget, store it, and send the budget out to the rest of the business.

Backup

Cloud computing is used for backing up devices when the data for a device is erased. Using the cloud, all of the pictures and documents from that device can be transferred seamlessly back on.
