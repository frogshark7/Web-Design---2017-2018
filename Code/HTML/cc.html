There are 3 Main Types of Cloud Computing

IaaS
PaaS

Infrastructure as a Service is when you rent a Infrastructure like a server, storage, operating systems, exedra from a wireless provider. Some IaaS examples are Servers and storage, Networking firewalls and security, a Data center, and a physical plant or building.
Platform as a Service is a bit broader than Infrastructure as a Service, since Infrastructure as a Service is a subsection of Platform as a Service. Platform as a Service is a Cloud computing service Based off of a computer platform, like google docs. It is used with operating systems, Development tools on a computer, Database management, Business analytics, and all of the uses listed under IaaS.
Finally, Software as a Service is a a service where something is downloaded as a software onto a device. This counts as apps, launchers, and all of the uses listed in Platform as a Service and Infrastructure as a Service.
