Public cloud market starts to level out

The public cloud market, while still prominent, is losing some popularity from the other cloud services. This is most likely because more and more people are starting to care about their personal information being accessed anywhere, so the private and hybrid cloud services are becoming more and more popular. More people are investing in the cloud

There is a steady increase of cloud computing use. This means that cloud computing is still popular and has not reached its peak in popularity yet. As more and more people use the cloud, the cloud traffic will grow to an estimate of 17.1 zetabytes (336,861,356,762 DVDs) in 2020.

The multi-cloud strategy is becoming more prominent

There is an increase in the multi cloud strategy and a decrease in the single cloud strategy. The multi cloud strategy is a strategy where two or more cloud computing services are used, so people don't have to be on a public cloud system all of the time. The multi cloud strategy has increased by 4 percent, while the Cloud system has decreased by 8 percent. With these statistics, some expect that the multi cloud system to become more and more prominent through 2020, and the single cloud system to do the opposite.

Market Growth

The Market Revenue in 2017 globally was about $180 billion. This number is growing at an annual rate of about 24% total. Iaas (Infrastructure as a service) and Paas (Platform as a service) are by far the fastest growing because of their popularity by Amazon and Microsoft, at 47% annual growth. Then, with a 31% growth rate, Microsoft and Salesforce lead the Saas (Software as a service) part of cloud computing. UCaas (Unified Communications as a service) has around a 20% growth rate along with surprisingly the private and public cloud. Iaas is growing really fast, and there are a few reasons why. First, Iaas is ready and available. Companies can quickly and easily rent out something on the cloud rather than buying a machine and then setting it up and replacing it. Best of all, Iaas has very low maintenance. Iaas is becoming more and more appealing. Paas is popular because lots of us are using internet-connected platforms like social media and video games. Predictions

Modular Software

Since the amount of hardware demanded by the cloud in the form of IaaS will increase, the amount of software required will follow. Modular software, where components of the software can be replaced without the program shutting down, will be of higher demand since individual program will only get larger and more complex. This is also due to cloud services being spread apart into the servers of different cloud services.

Software Drifts from Hardware

Some people argue that software will start to ‘float’ away from hardware. This is primarily due to a growing trend where businesses are using more cloud applications to automate business. Eventually this will result in a much more abstract environment where additional software has to go through several filters before interacting with hardware and so that applications running on the platform given by the provider will have never even known of the hardware.

Clouds will Become Specialized

Clouds, today, are differentiated based on whether they provide IaaS, PaaS, or SaaS. In the future, some predict that more specialized clouds will emerge to deal with new and existing tasks. This allows companies to move more specific tasks to the cloud rather than having an onsite infrastructure to deal with it.

Data Centers Become Ecosystems

With the progress of hardware and software, data centers are expected to become more automated. A central system that controls the hardware with software, will grow and shrink the amount of infrastructure dedicated to a specific workload dependent on the size. They’re also expected to patch and update equipment regularly, hence being comparable to a biological ecosystem.
