Viability

Cloud computing allows businesses to focus on their growth and success rather than focus on the complexities of maintaining and running their own IT services. This saves the business from buying additional hardware and services, allowing them to invest in more important things. Cloud computing also gives the users a platform that’s easily accessible, meaning that you don’t have to install or configure additional software. Plus, cloud computing is an important choice to consider for businesses in this day of age, since you can expand your businesses with the snap of a finger and improve its security as well. A study conducted by Exact, a software company, and Pb7 Research validated this, and found that small businesses who adopted cloud computing doubled their profits while seeing 25 percent of additional revenue growth. Cloud computing is extremely viable. It has survived due to its versatility, and because it provides existing infrastructure and services that improve the productivity of companies, instead of paying for costly IT services and hardware. Plus, companies only have to pay for the services they use from their provider.

Major Users

Apple
Apple has been with computing since the release of iTools in 2000, and has been using it ever since. Today, Apple uses cloud computing with its communication software, Mac.com, and has Apple pay, an app, which allows you to pay people wirelessly. Also, Apple uses a private cloud to store photos and documents transferred there from devices. Plus, Apple uses Cloud computing with its website hosting, and app store.

Google
Google is both a cloud computing creator and a major cloud computing user. In fact, it uses cloud computing more than it creates a new form of cloud computing. There uses are for Communication with google’s email site, gmail, Payment, through a wireless payment site called google pay. Also, Google uses Hybrid clouds with it’s google drive application, so your work is secure but can be shared to a public cloud if you want.

Facebook
Facebook is a communication software that connects people across the globe, and even this site uses Cloud computing. Facebook uses cloud computing for communication and website hosting, so the people using it are able to communicate with friends. Also, Facebook uses a hybrid cloud type storage system, where you can save pictures onto it, and release them on the site if you would like to.

Netflix
Almost every type of business is in the cloud, even a movie streaming company. Netflix uses cloud computing and cloud softwares that other companies have created, such as Amazon's AWS (which it is now leaving), and uses Apache code bundles for Netflix's Open cloud service. Netflix also uses scalability form the cloud ever since it realized it couldn’t keep up with the internet traffic.

Wells Fargo
Wells fargo is a banking company with multiple franchises in the cloud. For example, Wells Fargo has made apps that allows people to pay and store their credit cars wirelessly from their phones, using the cloud. Cloud computing gives Wells Fargo the infrastructure to maintain these apps and also gives security to its users.

Twitter
Like all social media platforms, Twitter uses the cloud for up to 95 percent of their communication and data analysis. The main use of cloud computing across all social media platform is to compare people’s information so as to build networks and connections. Twitter, specifically, uses apache to analyze and interpret the data.

Disadvantages

Flexibility

The cloud Has gained much support over the years due to many things, but most of all, because of its flexibility. This means that the files needed can be accessed anywhere, at any time, instead of at a certain place when that place is open.

Servers Could Go Down

When the servers go down, everything on the server is inaccessible. And since it is entirely random, a server crash could happen right before a project is due. To prevent all of the data from being lost, keep hard copies of your most important data, in case it happens.

Efficiency

Cloud computing is very efficient. This is because it creates less hassle and things go smoother. Also, cloud services are usually organised, so there aren't any problems finding things.

Hacking

The Cloud isn’t always secure and is vulnerable to hacking. To keep your personal data from being uncovered, have a unique password, and choose the cloud service that is the most secure server that is the safest. Also, be very vague on most of your data, and if there is a rather not say button, choose it.

Improved Collaboration

With the cloud, communication has become way easier. Now, people can communicate on websites, call others, and write emails. Because of all of this communication, more people collaborate on projects, and have the time to talk about the work.

Loss of Data Control

When people are moving services and software to the cloud, they end up giving all of their information to the company providing the cloud service. This can be a risky move for a company looking for a cloud provider since data on the cloud isn’t always secure.

Smaller Expenses

Cloud computing is very cost efficient because it deems buying hardware (servers, electricity) and IT services unnecessary. This, in turn, lowers the cost of cloud computing, and lowers the hassle as well. Because of this, cloud computing has become a cheap and useful source of computing.

Unreliable

Cloud computing isn’t always the most reliable. For example, when a cloud service is unresponsive, you can’t really do anything about it. To make sure that unreliability isn’t as bad of a problem, keep up with your cloud provider so you can prepare for any liabilities.
